Bitakati Mobile App

This is a mobile application developed using Flutter. It showcases a simple and intuitive interface to help users manage their tasks effectively. The app includes features such as task creation, editing, and deletion, as well as the ability to categorize tasks.

The app is still in the development phase, but it showcases the power of Flutter in creating cross-platform mobile applications with a smooth and responsive design.
