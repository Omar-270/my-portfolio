Jnnaty Web Page

About the Project

Jnnaty is a project that showcases a simple, informative web page for a concept of a smart city. The page highlights the city’s services, smart systems, and general information about the innovative features that it offers. The site also includes a contact form and user interaction elements.

This project was developed using **HTML** and **CSS** for layout and design. It aims to present clean and modern UI/UX design principles while also being informative and user-friendly.
